# CHAMPION-MASTERY-V4

## ChampionMasteryDto

プレイヤーとチャンピオンの組み合わせによる、シングルチャンピオンマスタリーの情報が含まれるオブジェクトです。

<Properties>
    <Property name="championPointsUntilNextLevel" type="long">
        次のレベルに到達するために必要なポイント数です。このチャンピオンのレベルが最大になった場合は0になります。
    </Property>
    <Property name="chestGranted" type="boolean">
        今シーズン、このチャンピオンにチェストが付与されたのか、されなかったのか。
    </Property>
    <Property name="championId" type="long">
        このエントリーのChampion ID
    </Property>
    <Property name="lastPlayTime" type="long">
        このチャンピオンがこのプレーヤーによってプレイされた最後の時間 - Unixミリ秒時間フォーマットで。
    </Property>
    <Property name="championLevel" type="int">
        指定された選手とチャンピオンの組み合わせのチャンピオンレベル。
    </Property>
    <Property name="summonerId" type="string">
        このエントリーのサモナーID。(暗号化)
    </Property>
    <Property name="championPoints" type="int">
        このプレーヤーとチャンピオンの組み合わせのチャンピオンポイントの総数です - チャンピオンレベルの決定に使用されます。
    </Property>
    <Property name="championPointsSinceLastLevel" type="long">
        現在のレベルを達成してから獲得したポイント数。
    </Property>
    <Property name="tokensEarned" type="int">
        現在のchampionLevelでこのチャンピオンが獲得したトークンです。championLevelが上がると、tokensEarnedは0にリセットされます。
    </Property>
</Properties>
