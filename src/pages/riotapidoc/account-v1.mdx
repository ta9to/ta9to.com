# ACCOUNT-V1

account-v1には、americas, asia, europeの3つのルーティング値があります。どの地域のどのアカウントに対してもクエリーを実行できます。最も近いクラスターを使用することをお勧めします。

---

## puuidでアカウントを取得する {{ tag: 'GET', label: '/riot/account/v1/accounts/by-puuid/{puuid}' }}

<Row>
    <Col>
        ### Required attributes
        <Properties>
            <Property name="ppuid" type="string" />
        </Properties>

        ### Return value
        AccountDto
        <Properties>
            <Property name="puuid" type="string">
            </Property>
            <Property name="gameName" type="string">
                このフィールドは、アカウントにgameNameがない場合、レスポンスから除外される場合があります。
            </Property>
            <Property name="tagLine" type="string">
                このフィールドは、アカウントにタグラインがない場合、レスポンスから除外されることがあります。
            </Property>
        </Properties>
    </Col>
    <Col sticky>

        ```bash {{ title: 'cURL' }}
        curl -G https://asia.api.riotgames.com/riot/account/v1/accounts/by-puuid/{ppuid} \
        -H "X-Riot-Token: {token}"
        ```

        ```json {{ title: 'Response' }}
        {
            "puuid": "****",
            "gameName": "****",
            "tagLine": "****"
        }
        ```
    </Col>
</Row>
